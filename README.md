# 9급 공무원(전산직) 영어 학습 앱

국가직 9급 공무원 영어 시험을 위한 **설치 없이 브라우저로 바로 쓰는** 학습 웹앱입니다.
[`index.html`](index.html)을 더블클릭하면 실행됩니다. (인터넷·서버 불필요, 진행상황은 브라우저에 자동 저장)

## 기능

### 📝 기출문제
- **2016~2025년 국가직 9급 영어 기출 200문항** 수록
- 연도 선택 → 20문항 풀이 → **공식 최종정답으로 자동 채점** + **문항별 해설**
- 정답/오답/미응답 표시, 연도별 최고점 저장

### 📘 어휘 (약 2,000단어)
- **단어덱**: 기출 36 + 빈출 핵심 1,840 + 실용·중급 168 + 생활영어 49 = **2,093개**
  - **Day 단위 학습**: 하루 N개(20/30/36/50)씩 묶어 진도 관리, "이어서 학습"으로 안 끝난 Day로 바로 이동
  - **🔊 발음 듣기**(음성합성), 예문, 카테고리 필터
- **🔁 복습(간격 반복·SRS)**: "외웠어요"는 점점 긴 간격으로, "헷갈려요"는 곧 다시 — 잊을 때쯤 자동으로 복습 대기열에 모임
- **어휘 기출문제**: 어휘 유형 기출 + 해설

### ✏️ 어법
- 어법·영작 기출문제 + 해설, 빈출 문법 개념 카드

### 📖 독해
- 유형별(대의파악·빈칸추론·순서/삽입·내용일치) 기출 + 해설

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `styles.css` | 스타일 |
| `app.js` | 앱 로직(기출/어휘/복습/어법/독해) |
| `kichul.js` | 기출 200문항(지문·보기·정답·해설) + 기출 어휘덱 |
| `vocab.js` | 추가 어휘(빈출 핵심·실용·생활영어) 약 2,000개 |
| `data.js` | 문법 개념 카드 데이터 |

## 데이터 출처 및 검증
- 기출 문제·정답은 **인사혁신처 사이버국가고시센터(gosi.kr)** 공개 자료(공식 문제지 + 최종정답표)에서 추출했으며, 정답은 공식 최종정답표 기준으로 교차검증했습니다.
- 해설·추가 어휘는 학습 보조용으로 생성되었습니다(시중 보카처럼 정교한 빈출 순위·검수가 된 것은 아니므로, 메인 어휘서와 병행 권장).

## 알려진 한계
- **어법 밑줄·문장 삽입** 유형은 보기가 지문 안에 표시됩니다(원본과 동일 구조).
- **옛 연도(2016~2021)** 일부 한글 보기는 원본 PDF 특성상 띄어쓰기가 붙어 보일 수 있습니다(영문·정답·해설은 정확).

## 비고
원본 기출/정답 PDF는 용량 문제로 저장소에서 제외되어 있습니다(`.gitignore`). 학습 데이터는 `kichul.js`·`vocab.js`에 모두 포함되어 앱 구동에는 지장이 없습니다.
