# 9급 공무원(전산직) 영어 학습 앱

국가직 9급 공무원 영어 시험을 위한 **설치 없이 브라우저로 바로 쓰는** 학습 웹앱입니다.
[`index.html`](index.html)을 더블클릭하면 실행됩니다. (인터넷·서버 불필요, 진행상황은 브라우저에 자동 저장)

## 기능

### 📝 기출문제 (메인)
- **2016~2025년 국가직 9급 영어 기출 200문항** 수록
- 연도 선택 → 20문항 풀이 → **공식 최종정답으로 자동 채점**
- 문항별 정답/오답/미응답 표시, 정답 강조, 연도별 최고점 저장

### 📘 어휘
- 빈출 영단어 플래시카드(예문 포함) + 영↔한 4지선다 퀴즈
- 외운 단어 체크, 틀린 단어 자동 복습 목록

### ✏️ 문법
- 빈출 문법 12개 포인트 정리 + 4지선다 문제·해설

### 📖 독해
- 유형별(주제·제목·빈칸·내용일치) 연습 지문 + 해설

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `styles.css` | 스타일 |
| `app.js` | 앱 로직(기출/어휘/문법/독해) |
| `kichul.js` | 기출 200문항 데이터(지문·보기·정답) |
| `data.js` | 어휘/문법/독해 학습 데이터 |

## 데이터 출처 및 검증
- 기출 문제·정답은 **인사혁신처 사이버국가고시센터(gosi.kr)** 공개 자료(공식 문제지 + 최종정답표)에서 추출했습니다.
- 정답은 공식 최종정답표 기준이며, 여러 연도를 직접 풀어 교차검증했습니다.

## 알려진 한계
- **어법 밑줄·문장 삽입** 유형은 보기가 지문 안에 표시됩니다(원본과 동일 구조).
- **옛 연도(2016~2021)** 일부 한글 보기는 원본 PDF 특성상 띄어쓰기가 붙어 보일 수 있습니다(영문·정답은 정확).

## 비고
원본 기출/정답 PDF는 용량 문제로 저장소에서 제외되어 있습니다(`.gitignore`). 학습 데이터는 `kichul.js`에 모두 포함되어 있어 앱 구동에는 지장이 없습니다.
